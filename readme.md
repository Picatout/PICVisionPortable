PICVisionPortable

console de jeux rétro portable.
basé sur un affichage LCD monochrome 128x64 pixels
machine virtuelle CHIPCON étendue.

Microcontrôleur:  PIC24FJ64GA002,  64Ko FLASH,  8Ko RAM
Affichage LCD passif  128x64 pixels.
sortie audio: 2 tonalités simultanées + bruit blanc.
